Credits are the currency for Gomobile services. Use these endpoints to monitor your balance and track usage.

## Check your balance

Get your current credit balance:

```bash
curl -X GET https://api.gomobile.ma/api/credits/balance \
  -H "x-api-key: YOUR_API_KEY"
```

Response:

```json
{
  "balance": 1500.00
}
```

## Transaction history

View your credit transaction history:

```bash
curl -X GET "https://api.gomobile.ma/api/credits/transactions?page=1&limit=20" \
  -H "x-api-key: YOUR_API_KEY"
```

Response:

```json
{

```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | number | 1 | Page number |
| `limit` | number | 20 | Items per page (max 100) |
| `startDate` | string | - | Filter from date (ISO 8601) |
| `endDate` | string | - | Filter to date (ISO 8601) |

### Filter by date range

```bash
curl -X GET "https://api.gomobile.ma/api/credits/transactions?startDate=2025-01-01&endDate=2025-01-31" \
  -H "x-api-key: YOUR_API_KEY"
```

## Transaction types

| Type | Description |
|------|-------------|
| `initial` | Initial credit allocation |
| `credit_add` | Credits added to account |
| `call_charge` | Deduction for call usage |
| `sms_charge` | Deduction for SMS campaign |
| `adjustment` | Manual adjustment |
